# Tugas Saat Ini

Sedang mengerjakan pembuatan dan perbaikan skrip `dev.ps1` untuk menjalankan lingkungan pengembangan (backend dan frontend) di Windows menggunakan PowerShell. 

Tujuannya adalah agar skrip dapat dijalankan dengan lancar dan semua path direktori dapat ditemukan dengan benar, serta memastikan virtual environment `.venv` dapat dibuat dan diaktifkan dengan baik.
